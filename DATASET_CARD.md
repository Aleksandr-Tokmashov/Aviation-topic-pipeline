# Dataset Card: Aviation Telegram Posts Dataset

## Описание датасета

### Краткое описание
Датасет содержит текстовые сообщения из 7 популярных Telegram-каналов, посвященных авиации, авиакомпаниям и аэропортам. Данные собраны за период 2022-2024 годов.

## Состав датасета

### Каналы-источники

| Канал | Описание | Сообщений |
|-------|----------|-----------|
| @aeroflot_official | Официальный канал Аэрофлота | 1,000 |
| @aviaComments | Новости и обсуждения авиации | 1,000 | 
| @s7newsroom | Новости S7 Airlines | 1,000 | 
| @favt_ru | Федеральное агентство воздушного транспорта | 4,000 | 
| @aviatorshina | Сообщество пилотов | 2,000 |
| @rumayday | Авиационные инциденты и безопасность | 1,000 |
| @aviationmode | Общие авиационные новости | 1,000 |
